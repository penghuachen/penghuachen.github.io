不過首先，我們先學 axios 這個套件的一些使用方式吧!

Axios 是什麼與如何使用

axios 是一套基於 promise 的 http 庫，是可以用來發送非同步請求的 library。

以往在前端開發時，常常會透過以下方式來發送非同步請求:

  • XMLHttpRequest
  • Fetch
  • jQuery 的 $ajax

然而由於 axios 的出現，讓我們在做非同步請求時，可以透過更簡單的方式來執行。

相關的知識點可以透過文件來理解，蠻容易懂的。

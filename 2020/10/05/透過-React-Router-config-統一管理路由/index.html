透過 React Router config 統一管理路由

前幾天我們學習到了如何在元件中使用 Router 來導航，而今天要學習的部分則是如何透過集中管理所有路由的方式實現導航到各個頁面的方式。

而這個方式就是在配置一個 router 的 config 檔案。

今天會透過兩個範例來理解這個部分:

  1. 只有一層路由的 Router 配置
  2. 具有巢狀路由的 Router 配置

接著就趕緊今天的進度吧!

Target #2 - Carrom

撰寫思維

  1. 設定四個 div 的區塊並使用 flex 語法調整到需求的位置

    • flex-wrap:wrap 此語法可以讓 flex-item(指的是 div)在容器寬度不夠時換行
    • justify-content:space-between 此語法可以讓 flex-item 在主軸的方向分配在容器中彈性元素之間及其周圍的空間。

  2. 透過 css selector 的 nth-child 和 ~ 這兩個選取器對於第二個 div 之後的所有 div 都做 margin-top:58px 的設定
